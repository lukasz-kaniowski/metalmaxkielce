# Website for Metal Max Kielce

## Features

* [Google structured data](https://developers.google.com/structured-data/local-businesses/)
* Google analytics
* Images on cloudinary

## Development

    $ npm i
    $ metalsmith

In new terminal start http server and point to *build* directory

    $ http-server build


### Hints:

* **Livereload** is used to ease development.
* If there is no http-server install it with ``` $ npm i -g http-server ```

## Deployment

    $ npm i
    $ metalsmith
    $ npm run deploy
